🗺️ 주소 검색 안내

주소 검색은 다음 우편번호 서비스를 사용합니다.
API 키 없이 바로 사용 가능하며, 정확한 한국 주소를 검색할 수 있습니다.

✅ 설정 완료 - 바로 사용 가능!

별도 설정 없이 주소 검색 기능을 사용하실 수 있습니다.
매물, 미팅, 동선, 지도 페이지에서 "검색" 버튼을 누르면 주소 검색 팝업이 열립니다.

사용 방법
  1. 매물/미팅/동선/지도에서 "검색" 버튼 클릭
  2. 팝업에서 주소 검색 (도로명, 지번, 건물명 모두 가능)
  3. 원하는 주소를 클릭하면 자동으로 입력됩니다
  4. 지도에서 핀 위치를 드래그하여 미세 조정 가능
