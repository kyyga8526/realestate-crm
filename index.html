주소 검색은 다음 우편번호 서비스로 바로 사용 가능합니다.
정확한 지도 위치를 위해 카카오 JavaScript 키를 등록하세요.

설정 방법
  1. developers.kakao.com → 부동산 CRM 앱 → 플랫폼 키
  2. JavaScript 키를 복사하여 위에 붙여넣기
  3. JavaScript 키 수정 → SDK 도메인에 현재 사이트 주소 등록
  4. "테스트 및 저장" 클릭하여 확인
